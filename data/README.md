# This folder contains small samples of the image dataset used.

Original Dataset Sources:

**Kaggle (3 classes : Healthy, Hispa, Brown Spot, Leaf Blast)**
https://www.kaggle.com/datasets/minhhuy2810/rice-diseases-image-dataset/data

**Kaggle (6 classes: Bacterial Leaf Blight, Brown Spot, Healthy, Leaf Blast, Leaf Scald, Narrow Brown Spot)**
https://www.kaggle.com/datasets/dedeikhsandwisaputra/rice-leafs-disease-dataset?resource=download 

**UC Irvine (3 classes: Brown Spot, Bacterial Leaf Blight, Leaf Smut)**
https://archive.ics.uci.edu/dataset/486/rice+leaf+diseases

**Mendeley Data (3 classes: Brown Spot, Bacterial Leaf Blight, Leaf Smut)**
https://data.mendeley.com/datasets/dwtn3c6w6p/1

Final classes that are chosen: 
**Healthy, Brownspot, Bacterial Leaf Blight & Leaf Blast**
